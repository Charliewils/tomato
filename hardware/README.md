# 硬件设计

## 传感器接线

### I2C 总线拓扑

```
i2c-3 (21120000.i2c) —— ADS1115 ADC @0x48 —— 电容式土壤湿度探头
                        (AIN0, ±4.096V, 128SPS)

i2c-4 (21130000.i2c) —— VEML7700 光照传感器 @0x10
                    —— SCD41 CO₂/温湿度传感器 @0x62
```

### VEML7700 光照传感器
| 引脚 | 连接 |
|------|------|
| VCC | 3.3V |
| GND | GND |
| SDA | i2c-4 SDA |
| SCL | i2c-4 SCL |

### SCD41 CO₂/温湿度传感器
| 引脚 | 连接 |
|------|------|
| VCC | 3.3V |
| GND | GND |
| SDA | i2c-4 SDA |
| SCL | i2c-4 SCL |

### ADS1115 + 土壤湿度探头
| 引脚 | 连接 |
|------|------|
| VCC | 3.3V |
| GND | GND |
| SDA | i2c-3 SDA |
| SCL | i2c-3 SCL |
| AIN0 | 电容探头模拟输出 |

标定：干端(空气) 2.772V → 0%，湿端(泡水) 1.540V → 100%

## 执行器接线

### GPIO 引脚映射

| 设备 | 丝印 | Linux GPIO编号 | 公式 |
|------|------|:---:|------|
| LED补光灯 | GPIO4_A0 | 128 | `4×32 + (1-1)×8 + 0 = 128` |
| 排风扇 | GPIO4_A1 (丝印GPIO23) | 129 | `4×32 + (1-1)×8 + 1 = 129` |
| 灌溉水泵 | GPIO3_B7 (丝印GPIO3_B7_D) | 111 | `3×32 + (2-1)×8 + 7 = 111` |

GPIO编号公式：`GPIO n_xy = n×32 + (x-1)×8 + y`（A=1, B=2, C=3, D=4）

### 通用隔离方案

```
GPIO (3.3V) ──→ [光耦模块 PC817] ──→ 执行器
                   ↑
              独立12V电源
              (与主板隔离)
```

### LED补光灯接线

```
GPIO128 (3.3V PWM 500Hz)
    │
    ▼
PC817 光耦模块 IN
    │  (V1=5V, 由12V经78L05降压)
    ▼
恒流模块 EN (2.3~5.6V)
    │
    ▼
LED灯条 (12V独立供电)
```

⚠️ 光耦跳线帽必须拔掉（否则输入/输出地共地→破坏隔离）
⚠️ V1只能接5V，严禁接12V

### 排风扇 + 水泵接线

```
GPIO129/111 (3.3V)
    │
    ▼
光耦继电器模块 IN
    │  (低电平触发)
    ▼
继电器触点 → 12V独立电源 → 设备
```

## 安全注意事项

1. **GPIO绝不直连12V侧负载** —— 必须经光耦/继电器隔离
2. **12V独立供电** —— 执行器负载电源与主板供电物理分离
3. **防反接** —— 12V输入端加防反接二极管或防呆接插件
4. **续流二极管** —— 感性负载（电机/泵/风扇）两端必须并联1N4007
5. **GPIO切换保护** —— 软件层≥0.3s最小间隔，防止浪涌叠加
6. **RC吸收器** —— 继电器触点建议并联100Ω+0.1µF吸收回路
7. **共地注意事项** —— 光耦两侧地线必须物理隔离，跳线帽必须拔掉
